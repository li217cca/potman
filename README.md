# Potman Beta 0.2

Potman is a chrome extension that helps to enjoy Granblue Fantasy easier.

## How to start?

### Install
1. 拉取项目至本地
```
git clone https://github.com/li217cca/potman.git
```
2. Chrome 右侧下拉菜单 > 更多工具 > 扩展程序
3. 扩展程序页面中打开右上角的开发者模式
4. 加载已解压的扩展程序 > 选中项目所在目录

### Update
```
git pull
```
